# greenvim

### What is greenvim?
greenvim is my attempt to make neovim as similar to an IDE as possible. I use lua for the config. 

### features:
* Completion using Coc.nvim
* Fuzzysearch
* bufferline
* theme
* tabnine
* github copilot
* Auto-pairs (brackets)
* auto indents
* file tree
* snippets
* better scrolling
* [full Unity support](#unity-support)
* [full Java support](#java-support)
